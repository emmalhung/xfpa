A.11.1. Template of a 'Constants' Block

Constants

{

constant_name

{

label = label

This is the label that the user sees. The default is constant_name. This parameter is "language" sensitive!

short_label = label

The short_label is used when there is not enough room on the FPA screen or button to use the longer label. The default is constant_name. This parameter is "language" sensitive!

description = label

This is an optional descriptive label. This parameter is "language" sensitive!

constant = value unit_name

This is the numeric value and units for constant_name.

} #(End of constant_name Block)

} #(End of Constants Block)
