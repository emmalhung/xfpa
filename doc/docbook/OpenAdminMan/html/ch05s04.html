5.4. Target Map Setup

The target_map block specifies the map projection information for displaying and building depiction charts. The following key words are recognized:

Note

Latitude and Longitude values may be specified in any of the following formats.

Whole degrees[ +/- ] DDD [ N/S/E/W ]
Decimal degrees[ +/- ] DDD.DDD [ N/S/E/W ]
Degrees, minutes, (seconds)

[ +/- ] DDD:MM [ :SS ] [ N/S/E/W ]

[ +/- ] DDD˚MM' [ SS" ] [ N/S/E/W ]

The default directions, with no signs present, are in degrees North for latitudes and in degrees East for longitudes.
