6.8.2. Header Keys

At the moment there is just one header key defined and this is models followed by the recognized model names as specified in the Winds section of the CrossRefs block in the Config file. 

[models]
  Vg_Msl
  Vr_Msl
  Vc_Msl
  Vx_Msl
