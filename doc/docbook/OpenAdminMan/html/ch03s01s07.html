3.1.7. Test the FPA software

To test the FPA software, Run the FPA depiction editor, as in: xfpa or xfpa -s setup_file. Where setup_file is the name of the setup file to be used. If omitted, the default setup file is used (see .fparc below).

If you do not have a database, you can use a demo database located in $FPA/demo/.

Example 3.6. Using a demo database to test FPA

>cd $HOME
>tar -xzf $FPA/demo.tgz
>xfpa -s av_atlantic

You should see the FPA window appear. To exit from the FPA depiction editor: select Field → Quit on the menu bar, near the upper left; a confirmation requester will then appear; select Yes and the FPA depiction editor will terminate.
