A.12.1. Template of a 'Units' Block

Units

{

unit_name

{

label = label

This is the label that the user sees. The default is unit_name. This parameter is "language" sensitive!

short_label = label

The short_label is used when there is not enough room on the FPA screen or button to use the longer label. The default is unit_name. This parameter is "language" sensitive!

MKS_equivalent = mks_unit_name

This is an MKS unit that unit_name can be converted to.

MKS_conversion = factor offset

These are the parameters needed to convert unit_name to its MKS equivalent according to the equation:

MKS value = (value - offset)/factor.

} #(End of unit_name Block)

} #(End of 'Units' Block)
