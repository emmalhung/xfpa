A.5.1. Template of a 'Fields' Block

Fields

{

element_name level_name

{

label = label

This is the label that the user sees in the FPA menus. The default is the level_name label followed by the element_name label. This parameter is "language" sensitive!

short_label = label

The short_label is used when there is not enough room on the FPA screen or button to use the longer label. The default is the level_name short_label followed by the element_name short_label. This parameter is "language" sensitive!

field_group = group_name

This parameter will group the FPA fields in the interface. The group_name is from the 'Fields' sub block of the Groups block. The default group is 'Miscellaneous'.

element_info = Default
