4.1.4. data

The data directory contains data which applies to all FPA implementations. It contains the following directories:

Table 4.8. data directory

namepurpose
commonContains common map overlays, etc.
map_dirContains a template database directory tree for a particular weather centre, including predefined map backgrounds and forecast area definitions
remoteOne suggested destination for downloaded GRIB data from CMC Could be a NFS mount point.
