3.2.1. Test the FPA database

To test your new FPA database: run the FPA depiction editor, xfpa [-s setup_file]. Where setup_file is the name of the setup file for the database you just created. If omitted, the default setup file is used (see .fparc above).

You should see the FPA window appear. It will hopefully be displaying the correct map background. If you just installed this database, there will be no depictions in the sequence. To exit from the FPA depiction editor: select Field → Quit on the menu bar, near the upper left; a confirmation requester will then appear; select Yes and the FPA depiction editor will terminate.
