13.4. Rotating a link chain

To rotate a link chain:

  1. Select the Move tool.

  2. Pick the link chain(s) to rotate. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Rotate.

    The picked link chain(s) turn blue.

  4. Click the left mouse button at the point you want to rotate around (the centre of rotation).

  5. Select a second point to form a handle to rotate the link chain(s) and hold the button down.

    A line displays between the two chosen points. This is your rotation handle (like the handle of a car window).

  6. Turn the handle to a new location.

  7. When you have reached the point where the rotation should end, release the left mouse button.

    The FPA displays the rotated link chain(s).

    Note: You can still restore the field if you click Undo at this point.

Note: The link chain(s) remain selected until you deselect them, so you may continue to use the Move tools. (see Unpick an object, (Section 9.4.2))
