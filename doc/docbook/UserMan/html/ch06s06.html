6.6. Updating issue times for guidance

A yellow box around the issue time for a guidance field in the guidance field list indicates that there is a more current issue time available.

To load the latest available issue time:

  1. Choose Guidance → Select... or click the icon in the "Command Icon Bar".

    The Guidance Field List dialog box appears. (see Figure 6.1)

  2. Click the Display tab.

  3. Select an Active List.

    Note: the issue time is Previous or Current you can check the actual time by clicking and holding the Issue display.

  4. Choose:

    Actions → Refresh Selected Fields

    To redraw the selected fields

    Actions → Set Selected to Current Issue Time

    Allow FPA to switch the display to the most current issue time when new guidance arrives.
