4.3.2. Displaying map overlays

To display one or more map overlays:

  1. Choose View → Map Overlay Control... or click the icon in the "Command Icon Bar".

  2. The Map Overlay Control dialog box appears. (see Figure 4.2)

    Figure 4.2. Map Overlay Control

  3. Click the radio buttons on or off to select the map overlays you want to display.

  4. Click Set to display the selected overlays.

  5. Click Close.

The Map Overlay Control may be left open while you work in the depiction editor. Normally a map metafile is read only once (when it is first displayed). If the contents of a map metafile change then a icon appears next to it.

To re-read modified map metafiles:

  1. Click Update.
