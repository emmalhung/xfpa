8.1.2. Placing text

To place text on your scratchpad:

  1. From the Function area, select Place Text. Scratchpad text menus appear (see Figure 8.2)

    Figure 8.2. Scratchpad text menus

  2. Select the text size, font type and colour for the text.

    Note: The point sizes are relative; if you resize a window, the size of the text scales accordingly.

  3. Click Set.

    An entry dialog box appears.

  4. Type your text.

  5. Click Set.

  6. Click where you want the text to appear on the scratchpad.

    The text appears, centred at the point where you clicked.
