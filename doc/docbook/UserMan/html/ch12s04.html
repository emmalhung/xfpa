12.4. Deleting an area

To delete an area:

  1. Select the Modify tool.

  2. Pick the area to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Delete Area.

  4. To restore the area, click the right mouse button and select Undo.

To delete multiple areas:

  1. Select the Move tool.

  2. Pick the area(s) to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Cut.

  4. To restore the area(s), click the right mouse button and select Undo or Paste.
