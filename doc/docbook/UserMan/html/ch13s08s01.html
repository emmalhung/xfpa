13.8.1. Moving a link node

To move a link node:

  1. Select the Edit Node tool.

    The Edit Node menus appear. (see Figure 13.4)

    Figure 13.4. Link node edit menus

  2. Select the Move option.

  3. Select a link chain to work on. The track turns yellow, and the nodes blue.

  4. Click the left mouse button on the node you want to move. Hold the mouse button down and drag the node to a new location. FPA will recalculated the field and display the resulting changes.

    Only Normal and Control type nodes may be moved. The position of Interpolated and Floating type node positions are calculated by FPA

  5. You may repeat the previous step on all Normal and Control type nodes on the selected link chain. Click the right mouse button and select Cancel to complete the action and deselect the link chain.

    Note: You can still restore the link chain if you click Undo at this point.
