15.2. Deleting points

To delete a point:

  1. Select the Modify tool.

  2. Pick the point to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Delete.

To delete multiple points:

  1. Select the Move tool.

  2. Pick the point(s) to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Cut.

  4. To restore the line(s), click the right mouse button and select Undo or Paste.
