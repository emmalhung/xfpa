10.4. Changing the value at a given point (Poke)

The Poke tool allows you to pick a point on the depiction and change its value. The field within the radius of influence is altered to smooth the resulting field. In the case of pressure, you can change the pressure value at a given point by a delta amount to deepen a low or increase a high.

To change the value at a given point:

  1. Select the Poke tool.

    The Poke menus appear. (see Figure 10.4)

    Figure 10.4. Continuous field edit poke menus

  2. Move the slider on the slider-bar to adjust the delta amount. (see Using slider-bars, (Section 3.3.5))

    Positive values increase the value at the specified point by that value and negative values decrease the value by that amount. (If you select zero, nothing happens.)

  3. Select a radius of influence. (see Figure 10.3)

    The percentage indicates the affected area surrounding the point of edit.

    The cursor appears with the chosen radius of influence while inside the map area.

  4. Pick the point on the depiction you want to change.

    The FPA recomputes the field and re-displays the new field contours in white. The original contours remain in the normal field colour.

Remember to select your delta value first. If you forget and select the point to change first, the existing value is applied.
