6.3.2. Creating a new, empty guidance field list

To create a new list:

  1. From the menu in the Guidance Field List dialog box, select Lists → New....

    The Create List dialog box appears. (see Figure 6.6)

  2. In the List Name field, type the name of the new list.

  3. Click Apply.

    Now you can add fields to this list. See Adding fields to a guidance field list, (Section 6.3.3).
