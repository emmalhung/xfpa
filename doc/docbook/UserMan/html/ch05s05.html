5.5. Saving (and retrieving) depictions and fields

You can choose to save the active depiction or time sequence or save all depictions and fields.

  1. Choose Fields → Save Active or Fields → Save All.

To retrieve saved fields:

  1. Choose Fields → Load... or click the icon in the "Command Icon Bar".

    The Load Fields dialog box appears. (see Figure 5.1)

  2. Click the tab for: Normal, Daily or Static fields.

  3. Choose Backup as the Source.

  4. Select a field. (see Loading fields, (Section 5.1))
