6.7. Removing fields from guidance field lists

Note: You cannot remove a field from a predefined list.

To remove a field from a guidance field list:

  1. Choose Guidance → Select... or click the icon in the "Command Icon Bar".

    The Guidance Field List dialog box appears. (see Figure 6.1)

  2. In the Guidance Field List dialog box, select Fields → Remove....

    The Remove Fields From List dialog box appears. (see Figure 6.9)

    Figure 6.9. Remove Field From Guidance

  3. Select one or more fields to remove. Click once to select a field click again to deselect it.

  4. Click Remove.
