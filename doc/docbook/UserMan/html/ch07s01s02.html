7.1.2. Satellite

To display Satellite imagery:

  1. Choose Imagery → Select or click the icon in the "Command Icon Bar".

    The Image Control dialog box appears. (see Figure 7.1)

  2. Choose a satellite product from the drop-down list.

  3. Select satellite sources from the list. (see Selecting from a list , (Section 3.3.2))

    Note that multiple satellite images can be displayed at the same time. The image nearest to the top of the list is displayed on top of the others.

  4. Choose an Enhancement colour scheme for the satellite images from the drop-down list.
