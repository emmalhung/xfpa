9.1. Accepting, undoing and canceling an action

When you perform an edit, Accept and Undo become available. At this point, you may undo the last edit by clicking Undo or accept the last edit by clicking Accept. Once accepted, this edit cannot be undone.

Otherwise, if you proceed with some other action (another edit, change fields, change time, exit, etc.) the last edit is automatically accepted.

Some edits are performed in several steps (e.g., the Drag tool). In these cases, Cancel becomes available after the first step. This gives you the chance to cancel the current operation. Depending on the type of edit, Cancel causes the FPA to go back to the previous step, rather than canceling the edit entirely.
