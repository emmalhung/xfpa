13.2. Deleting link chains

To delete a link chain:

  1. Select the Modify tool.

  2. Pick the link chain to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Delete.

To delete multiple link chains:

  1. Select the Move tool.

  2. Pick the link chain(s) to delete. (see Pick an object, (Section 9.4.1))

  3. Click the right mouse button and select Cut.

  4. To restore the link chain(s), click the right mouse button and select Undo or Paste.
