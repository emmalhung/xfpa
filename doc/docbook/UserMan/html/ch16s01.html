With the FPA, you can use the Time Link feature to track significant locations through time. The links are used to indicate movement of features in the depictions and allow the depictions to be interpolated through time.

For a chosen field, you select a point on the field which identifies a common feature in each depiction; then you select a point identifying the same feature at a different time. The feature is now linked between the depictions.

Any feature on one field which is common to a feature on the same field at a consecutive time (either before or after) may be linked.

Note: It is the shape of the feature, not its value, that is important.

You can link depictions forward in time or backwards. (See Linking forward, (Section 16.2.1) and Linking backward, (Section 16.2.2).)
