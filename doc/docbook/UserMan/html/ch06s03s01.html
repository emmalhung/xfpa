6.3.1. Copying an existing list to a new guidance field list

To copy an existing field list (including the temporary field list):

  1. In the Guidance Field List dialog box, select an Active List to display.

  2. Choose Lists → Save as....

    The Create List dialog box appears. (see Figure 6.6)

    Figure 6.6. Guidance Create List

  3. In the List Name field, type the name of the new list.

  4. Click Apply.

    The contents of the new list automatically appear in the Display tab.

    Now you can add fields to this list. See Adding fields to a guidance field list, (Section 6.3.3).
