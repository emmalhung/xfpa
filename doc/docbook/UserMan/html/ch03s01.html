3.1. Tour of FPA interface

Figure 3.1. FPA Graphical Editor interface

Elements of the FPA Graphical Editor interface

  1. Main Menu.

  2. Lat/Lon co-ordinates of cursor on map. (see General Preferences, (Section 4.7.1))

  3. The command icon bar are short-cut buttons to frequently used menu items. Hover the mouse over an icon to see which menu item it launches. (see General Preferences, (Section 4.7.1))

  4. Depiction Sequence select time of working depiction. Part of the time selection bar (see General Preferences, (Section 4.7.1)). (see Defining T0 depiction, (Section 4.4) and Limiting the start and end time in the time sequence, (Section 4.6)),

  5. Forward/backward arrows step through the depictions. Part of the time selection bar (see General Preferences, (Section 4.7.1)).

  6. Date and time of the currently selected depiction. Part of the time selection bar (see General Preferences, (Section 4.7.1)).

  7. Message bar displays context sensitive information. (see General Preferences, (Section 4.7.1))

  8. Graphical depiction editing window.

  9. Side panel provides access to database editing tools. (see chapters 8 through 16.)
