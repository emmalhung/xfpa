6.3.3. Adding fields to a guidance field list

Note: You can not add a guidance field to a Predefined type list.

To add a field to a guidance field list:

  1. Choose Guidance → Select... or click the icon in the "Command Icon Bar".

    The Guidance Field List dialog box appears. (see Figure 6.1)

  2. Click the Display tab.

    You can either put guidance fields in a temporary guidance field list or create a new list.

    Note: If you choose the * Temporary * field list the FPA DOES NOT save the list for the next run of the FPA.
