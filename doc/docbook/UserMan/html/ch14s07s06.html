14.7.6. Changing the line type

To change the type of line:

  1. Select the Modify tool.

    The Modify menus appear. (see Figure 14.3)

  2. Pick the line to modify. (see Pick an object, (Section 9.4.1))

  3. From the line list, pick a line type.

    The line is redrawn as the selected type.
