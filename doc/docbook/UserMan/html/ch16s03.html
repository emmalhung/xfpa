16.3. Interpolating the linked depictions

The FPA can interpolate the set of depictions to smaller time increments (e.g., 1 hour) using the links you have specified.

To interpolate linked depictions:

  1. Click the Interpolate tab. (see Figure 16.17)

    Figure 16.17. Interpolation menus

    Note: The coloured boxes within the Group and Fields boxes. Each colour represents the state of the links in that area. (see Figure 5.11)

  2. To interpolate, click either Selected Fields or All Fields.

    If you choose All Fields the interpolation begins.

    If you choose Selected Fields, a Field Selection dialog box appears allowing you to choose which field to interpolate. When you have selected which fields you want to interpolate, click Accept to begin the Interpolation.
