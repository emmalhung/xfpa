14.1. Drawing lines

To draw a line:

  1. Select the Draw tool.

    The Draw menus appear. (see Figure 14.2)

    Figure 14.2. Line draw menus

    The cursor appears as a pencil while inside the map area.

  2. Select a Line Type from the list.

  3. Draw the line. (see Drawing with the "Freehand" tool, (Section 3.4.2) or Drawing with the "Curve" tool, (Section 3.4.3))

    The selected line type appears.
