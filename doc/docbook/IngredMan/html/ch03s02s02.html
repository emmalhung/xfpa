3.2.2. ARGUMENTS

type

The type of the message. Normally the type is used to index into the resource file to obtain information on the colour and font that the message should be displayed in. There is a special case where type is the keyword "ErrorDialog". In this case the interface is expected to display the message in a separate dialog.

text

Normally the text of the message. In the special case where type is "ErrorDialog", text contains the keyword into the CommonMdb file which contains the actual message.
