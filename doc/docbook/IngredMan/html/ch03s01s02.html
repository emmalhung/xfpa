3.1.2. ARGUMENTS

type

Specifies the type of cursor requested. The types send by ingred are:

"busy"

A cursor which indicates that the program is busy.

"finger"

A pointing finger.

"obscured"

A cursor which indicates that the drawing window is obscured.

"pen"

The figure of a pen or pencil.

"stop"

A stop sign.

state

Specifies the display state of the cursor. May be either on (True) of off (False).
