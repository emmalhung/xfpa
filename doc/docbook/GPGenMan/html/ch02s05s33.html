2.5.33. @gpgen_group_end

End forced grouping of objects for an output graphics file.

Note that this directive is not used by PSMet!

Note that this directive has no keywords.

Example 2.52.  

@gpgen_group_end { }

Ends grouping of graphic displays that began with @gpgen_group_begin directive.
