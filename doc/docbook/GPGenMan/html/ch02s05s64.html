2.5.64. @version

Set the PSMet or SVGMet version number.

Note that this must be the first directive!

VALUE

 

DESCRIPTION

 

psmet8.0

 

Current version number.

(Note: psmet6.0 is also accepted for backwards compatibility

 

svgmet8.0

 

Current version number.

(Note: svgmet6.0 is also accepted for backwards compatibility

 

Example 2.99.  

@version { psmet8.0 }

Set PSMet version number. Note that this must be the first directive!
