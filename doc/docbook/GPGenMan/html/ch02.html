2.5. Complete Description of All PSMet/SVGMet Directives

The PSMet and SVGMet applications allow the user to develop graphical products using data from the FPA. The graphical products can be customized to display the data on different geographical backgrounds, in different formats, and with various patterns, symbols, and headers. This section describes the PSMet and SVGMet files and commands in more detail.
