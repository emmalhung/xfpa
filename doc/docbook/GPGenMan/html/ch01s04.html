1.4. Format of Values in a Range

Several keywords in the Graphics Product Generator applications accept minimum and maximum values in a special format that allows for inclusive or exclusive ranges. These are identified by a keyword type of range. This format is given by:

[ min , max ]for a range min ≤ value ≤ max
( min , max ] for a range min < value ≤ max
[ min , max ) for a range min ≤ value < max
( min , max ) for a range min < value < max

A colon (:) may be used in place of the comma (,).

An asterisk (*) may be used in place of a min or max value; it is interpreted as an unlimited end of a range.

The following are examples of range formats:

Example 1.1. defines a range 75 < value ≤ 105

( 75 , 105 ]

Example 1.2. defines a range value < 105

[ * : 105 )
