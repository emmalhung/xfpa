2.5.21. @distance_scale_ticks

Display tick marks on a named distance scale.

KEYWORD

TYPE

DESCRIPTION

 

scale_name

string

Scale name (from @draw_distance_scale)

 

tick_units

string

FPA units (default is scale_units from @draw_distance_scale)

 

tick_length

real

length of tick in tick_units

 

Note that tick_location can be specified more than once in this directive!

 

tick_location

real

distance along the scale to place tick line in tick_units

 

tick_rotation

real

Angle of rotation (in counter-clockwise decimal degrees)

 

tick_justification

string

Tick justification

One of:left centre right

 

PRESENTATION KEYWORD

TYPE

DESCRIPTION

 

line_width

real

Line width for boundary (in display units)

 

line_style

string

Line style (see @presentation)

 

outline

string

Line outline colour (see @presentation)

 

Example 2.34.  

@distance_scale_ticks
{
scale_name          = distance_scale1;
tick_location       = 100;
tick_location       = 200;
tick_location       = 300;
tick_location       = 400;
tick_length         = 15;
tick_units          = km;
tick_justification  = left;
tick_rotation       = 90.0;
outline             = X11 Black;
line_width          = 0.03;
}

Add tick lines 100km, 200km, 300km and 400km along the scale. The tick length is 15km relative to the length of the scale. They are left justified on the scale and rotated to be perpendicular to the scale.
