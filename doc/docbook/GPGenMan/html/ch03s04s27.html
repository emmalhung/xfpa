3.4.27. @version

Set the TexMet version number.

Note that this must be the first directive!

VALUE

 

DESCRIPTION

 

texmet8.0

 

Current version number

(Note: texmet6.0 is also accepted for backwards compatibility

 

Example 3.36.  

@version { texmet8.0 }

Set TexMet version number. Note that this must be the first directive!
