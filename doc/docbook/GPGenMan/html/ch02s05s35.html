2.5.35. @group

Set a list of keywords to be included as a "macro".

Note that the keywords are included in another directive by using the format: <group_name in the keyword list.

KEYWORD

TYPE

DESCRIPTION

 

group_name

string

Group name for keyword list

 

acceptable_keywords

Any keywords that are acceptable in the directives that this "macro" is included within can be set here

 

Example 2.54.  

@group 
{
  group_name       = MaxTempAreas;
  element          = max_temp;
  level            = surface;
  units            = degreesC;
  display_as_areas = yes;
  outline          = none;
  fill             = none;
}

Set a "macro" called "MaxTempAreas" containing keywords used in the @contours directive, given in the example above.
