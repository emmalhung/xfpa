2.5.54. @resolution

Set the spline resolution for "continuous" or "vector" ﬁelds for the current map.

Note that map_units is automatically reset to the value of map_units in the @mapdef directive, and res is scaled accordingly.

Note that values determined by xmax - xmin and ymax - ymin keywords in the @mapdef directive should be multiples of the the value of the res keyword, given below.

KEYWORD

TYPE

DESCRIPTION

 

res

real

Spline resolution (in units of map_units keyword below)

 

map_units

real

Number of meters (or degrees) per map unit

(Default is map_units from @mapdef directive)

 

Example 2.80.  

@resolution { res = 200; map_units = 1000; }

Set a spline resolution for "continuous" or "vector" ﬁelds to 200 distance units, where all distance units are expressed in 1000 m, or km.
