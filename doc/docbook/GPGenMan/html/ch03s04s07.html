3.4.7. @group

Set a list of keywords to be included as a "macro".

Note that the keywords are included in another directive by using the format: <group_name in the keyword list.

KEYWORD

TYPE

DESCRIPTION

 

group_name

string

Group name for keyword list

 

acceptable_keywords

Any keywords that are acceptable in the directives that this "macro" is included within can be set here

 

Example 3.7.  

@group 
{
  group_name       = sample_winds;
  wind_crossref    = Fpa_Wind;
  units            = knots;
  format           = wind_text;
  table_name       = T1;
  x_off            = 1;
}

Set a "macro" called "sample_winds" containing keywords used in the @sample_wind directive, given in the example below.
