1.17. Graphics Wind Barbs

The PSMet and SVGMet applications use various keywords to allow winds or vector winds to be displayed as wind barbs. The directives that display wind barbs contain the format or attribute_format keyword, with a value of wind_barb. The keywords used to set the wind barb display parameters are found in the @wind_barb_presentation directive.

Examples of the display of wind barbs for the PSMet and SVGMet applications are shown in Appendix F. Appendix F also shows how the various keywords affect the display of wind barbs.
