17.3. Running Allied Models

The available models are determined by your System Administrator. This button is greyed out and not selectable if there are no models available to run.

To run Allied Models:

  1. Choose Products → Allied Models... or click the icon in the "Command Icon Bar".

    The Allied Model Run Select dialog box appears. (Figure 17.4)

    Figure 17.4. Allied Model Run Select Dialog

  2. Select the models to be run.

  3. Click Run.
