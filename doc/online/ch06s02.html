6.2. Checking available guidance fields

To check available fields:

  1. Choose Guidance → Availability.

    The Guidance Availability dialog box appears. (see Figure 6.4)

    Figure 6.4. Guidance Availability

  2. You can use the Level Filter to selectively display guidance field availability information.

    Figure 6.5. Guidance Availability - Level Filter

    If you select a Level Filter, only the fields from the selected level will be displayed in the Fields Available list. If you select None, all available fields will be displayed.

  3. Select Source.

  4. Select Issue Time.

    The Fields Available list displays the fields available from the selected Source and Issue Time.

  5. Click Close.
