3.2.3. Editing text

You can edit text in a number of areas in the FPA such as the weather entry menu or the problem report area.

Position the cursor

Click in the desired spot.

Select a word

Double-click in the word. The word is highlighted.

Select a line

Triple-click in the line. The line is selected.

Select a block

Click four times in the block (paragraph). The block is selected.

Replace text

Click and hold to highlight the text to replace. Start typing and the text is replaced.
