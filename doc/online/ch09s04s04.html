9.4.4. Select all objects in an outline

  1. Click the right mouse button and select:

    Select by → Draw an outline

    Use a drawing tool to draw an outline around the objects to be selected. (see Drawing with the "Freehand" tool, (Section 3.4.2) or Drawing with the "Curve" tool, (Section 3.4.3))

    Select by → Use last drawn outline

    Reuse the last drawn outline to select objects.

    Select by → Predefined outline

    Your System Administrator may configure some predefined outlines.
