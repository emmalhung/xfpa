4.7.3. Setting dialog locations

You can set the location for various dialogs if you have a multi-headed system.

  1. Choose Options → Preferences... and select the Dialog Location tab.

  2. Select which monitor a dialog should appear on.

  3. Click Apply to register your changes.

  4. Click Close.

Note: The alignment of the columns should correspond to the locations of your monitors. Contact your System Administrator if this is not the case.
