7.2.5. Radar Data View

If your radar images do not already contain range rings, they can be added using the Radar Data View options.

No Range Rings

Do not display range rings.

Limit Ring Only

Display a ring which displays the limit of the radar range only.

All Rings

Display the limit ring as well as rings spaced according to the Spacing spin-box.
