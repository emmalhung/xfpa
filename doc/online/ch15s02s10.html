15.2.10. Removing a node

You can remove the first and last nodes in a link chain as well as control nodes with the Delete Node tool.

To remove a node from your link chain:

  1. Select the Remove Node tool.

  2. Pick the link node you want to remove.

  3. The link node is removed.

Note: The Undo is available if you accidentally delete the wrong node.
