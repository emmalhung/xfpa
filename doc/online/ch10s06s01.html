10.6.1. Tightening a gradient

The Drag tool can be used to tighten a gradient or to extend a trough or ridge. You can use radius of influence (see Figure 10.3) to control the area of effect of the edit tool.

To tighten a gradient:

  1. Specify one set of drags that anchors one line by picking a point on the line and releasing the button in the same place.

  2. Specify a second set of drags that moves the second line towards the first.

  3. Click the right mouse button and select Proceed to indicate that you have completed the selection.

    The gradients are tightened.

    Note: The numerical solution to multiple drags is non-linear. Predicting the effect of multiple drags comes with experience.
