15.2.6. Creating and using a set of master links

To create a set of master links:

  1. From the Field list, select Group Master.

  2. Link forward or backward as you would for a field. (See Linking forward, (Section 15.2.1) or Linking backward, (Section 15.2.2).)

    See Creating and using a set of master links, (Section 15.2.6).

To use master links:

  1. From the Field list, select the continuous field you wish to link.

  2. From the Use Links From: option, select Group Master.
