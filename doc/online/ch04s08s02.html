4.8.2. Saving a profile

To save the active profile:

  1. Change the layout and interface options to suit your new requirements.

  2. Choose Options → Save Profile....

    The Save Profile dialog box appears. (see Figure 4.8)

    Figure 4.8. Save Profile dialog

  3. Set the Automatically update profile on exit from program option to your desired behaviour. (see Creating a profile, (Section 4.8.1))

  4. Click Save.
