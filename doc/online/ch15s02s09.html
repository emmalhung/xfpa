15.2.9. Adding a control node

You can add a control node to your link chain which you can use to change the link chain path.

Note: Control nodes can only be used in discrete area or line fields.

To add a control node:

  1. Select the Move Node tool.

    The Move menus appear. (see Figure 15.15)

    Figure 15.15. Move menus

  2. Select the Add Control Node checkbox.

  3. Pick the link chain you want to add a control node to. The picked chain turns yellow.

  4. If the links are too close together, a list of possible control times is displayed.

    Choose the time at which you want to add the control node.

  5. Click the link you want to become the control node and hold the button down.

  6. Drag the link to its new position and release the button.

  7. The link chain is redisplayed.
